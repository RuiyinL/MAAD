The Non-functional Requirements are:

1. **NFR-001 (Compatibility)**:
   - *Original text*: "The Space Fractions system requires a web browser capable of running Flash movies."
   - *Related quality attribute(s)*: Compatibility, Interoperability.

2. **NFR-002 (Hardware Constraints)**:
   - *Original text*: "The Space Fractions system does not require any new hardware."
   - *Related quality attribute(s)*: Portability, Resource Utilization.

3. **NFR-003 (Performance)**:
   - *Original text*: "Due to the relatively small size of the introductory and main menu movies, they can be downloaded in approximately one minute with a modem connection. Because Flash movies do not have to be fully downloaded to play, the main system can be played within a few minutes with a regular modem connection to the Internet."
   - *Related quality attribute(s)*: Performance, Efficiency.

4. **NFR-004 (Security)**:
   - *Original text*: "The Space Fractions system will be as secure as the web browser that will run the product."
   - *Related quality attribute(s)*: Security.

5. **NFR-005 (Reliability)**:
   - *Original text*: "Reliability will be ensured by extensive testing by the team members and mentors, if available."
   - *Related quality attribute(s)*: Reliability, Testability.

6. **NFR-006 (Maintainability)**:
   - *Original text*: "Maintainability is a primary goal for this project."
   - *Related quality attribute(s)*: Maintainability.

7. **NFR-007 (Usability)**:
   - *Original text*: "The information and interface will be effective so that Bobby will easily recognize what to do to start the system immediately and Alice will have no problems navigating through the help section to understand the rules and systemplay."
   - *Related quality attribute(s)*: Usability, Accessibility.

8. **NFR-008 (Availability)**:
   - *Original text*: "The Space Fractions system will be available over the Internet via the S2S website."
   - *Related quality attribute(s)*: Availability, Accessibility.

9. **NFR-009 (Data Integrity)**:
   - *Original text*: "The administrator of the Space Fractions system may wish to design a custom system complete with custom fraction questions. This information must be saved in a file on the web server [...] and will be easily edited through simplified administrative screens. [...] The tool then performs a validation check on the submitted data to ensure completeness and contextual relevance to the game's educational objectives."
   - *Related quality attribute(s)*: Integrity, Maintainability.

10. **NFR-010 (Scalability)**:
    - *Original text*: "Only one person can use a single instance of the Space Fractions system. However, the Space Fractions system will reside on the Internet so more than one user can access the product and download its content for use on their computer."
    - *Related quality attribute(s)*: Scalability, Concurrency.
